## Ключевые выводы

1. **Regular массивы стабильно быстрее** для `push` (особенно на больших размерах)
2. **`unshift` — катастрофически медленный** (в 1000+ раз медленнее `push`/`pop`)
3. **JIT-прогрев даёт ускорение** для большинства операций (~30-40%)
4. **`shift` на больших массивах (100k) без JIT работает ужасно** (медленнее в 1000 раз)
5. На маленьких массивах разница между Holey и Regular минимальна или отсутствует

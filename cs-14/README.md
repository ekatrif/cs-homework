## indexOf и lastIndexOf через бинарный поиск

Реализованы функции indexOf и lastIndexOf через бинарный поиск в соответствии с [заданием](./task.md).

```
node --experimental-strip-types functions.ts
```

## Отличия реализованных функций от нативных
- Алгоритмическая сложность нативных методов indexOf / lastIdexOf составляет O(n), в то время как сложность реализованного алгоритма через бинарный поиск - O(log n).
- В нативных методах нет требований к исходному массиву, в то время как реализованные функции используют отсортированный массив.
